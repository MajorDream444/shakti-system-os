# Phase 02 — Live Vault

## Scope

- Create or confirm Airtable base.
- Confirm Google Drive root folder.
- Map reviewed Drive assets into Airtable `Library Assets`.
- Add live sync behind explicit environment configuration.
- Add human review status before app exposure.

## Non-Goals

- No payment flow until access and content metadata are stable.
- No authentication redesign.
- No visual redesign.
