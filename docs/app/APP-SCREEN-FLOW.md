# Shakti Shala App Screen Flow

## Purpose

This document defines the first-pass screen flow for the Shakti Shala Mountain Path MVP.

## Screen 01 — Opening Threshold

### Intent

Create a calm first threshold into the app.

### Visual

Dark Himalayan sanctuary gate, ember light, stone, mist, quiet mountain atmosphere.

### Copy

```text
Shakti Shala

A mountain path for practice, rhythm, shadow, and Shakti.

You do not need to climb the whole mountain today.
Begin with the next steady step.
```

### CTA

```text
Begin the Path
```

## Screen 02 — Meet the Sherpa

### Intent

Introduce the Sherpa model clearly.

### Copy

```text
Your Shakti Shala Sherpa helps you ascend the mountain.

The Sherpa is not a guru.
The Sherpa reads the terrain, protects the pace, and helps you choose the next safe doorway.
```

### CTA

```text
Continue
```

## Screen 03 — Entry Discernment

### Intent

Gather first readiness signals.

### Questions

1. What are you carrying right now?
2. What pace feels supportive?
3. What are you longing for?
4. What kind of support feels sustainable?
5. What doorway are you most curious about?

### CTA

```text
Reveal My First Step
```

## Screen 04 — Mountain Map

### Intent

Show the digital pilgrimage structure.

### Visible Gates

- Base Camp
- Threshold
- ABC Path
- Moon Gate
- Durga Gate
- Kali Gate
- Lakshmi Gate
- Shakti Shala

### Sherpa Note

```text
The mountain is not climbed by force. Your path opens through rhythm, practice, and readiness.
```

## Screen 05 — ABC Path

### Intent

Introduce Tantra micro-teachings.

### MVP Letters

- A — Awareness
- B — Breath
- C — Chaitanya
- D — Dharma
- E — Embodiment

### Lesson Template

```text
Teaching
Reflection
Body Practice
Journal Prompt
Sherpa Note
```

## Screen 06 — Moon Gate

### Intent

Introduce rhythm before intensity.

### Flow

```text
Set intention
Practice for 3 days
Reflect
Release
Receive next prompt
```

### Sherpa Note

```text
Rhythm is how the body learns to trust the path.
```

## Screen 07 — Durga Gate

### Intent

Foundation, protection, boundaries, steadiness.

### Sherpa Note

```text
Durga protects the path before the deeper gates open.
```

## Screen 08 — Kali Gate

### Intent

Shadow, truth, death-rebirth, cutting illusion.

### Sherpa Note

```text
The Kali Gate is not closed. It asks for preparation.
```

## Screen 09 — Lakshmi Gate

### Intent

Receiving, worth, nourishment, beauty, value.

### Sherpa Note

```text
Receiving is also a practice of steadiness.
```

## Screen 10 — Pathway Reveal

### Intent

Recommend one aligned next doorway.

### Possible Results

- Open Gate
- Moon Rhythm
- Weekly Circle
- 1:1 Support
- 6-Week Container
- Retreat Readiness
- Temple Library

### Copy Pattern

```text
Your path points toward [DOORWAY].

Sherpa Note:
[Short reflection based on readiness signals.]

Next step:
[CTA]
```

## Screen 11 — Save My Path

### Intent

Capture contact information after the seeker has received value.

### Fields

- First name
- Email
- Optional WhatsApp

### CTA

```text
Send My Path
```

## Screen 12 — Rest Point

### Intent

End with integration, not pressure.

### Copy

```text
This threshold is complete.
Rest before ascending.
```
