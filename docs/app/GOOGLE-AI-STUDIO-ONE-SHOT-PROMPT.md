# Google AI Studio One-Shot Prompt — Shakti Shala Mountain Path MVP

Use this prompt in Google AI Studio to generate a first visual prototype of the Shakti Shala Mountain Path MVP.

---

## Prompt

You are creating a mobile-first web app prototype called **Shakti Shala: Mountain Path**.

This is not a generic wellness app, a game, a chatbot, or a guru simulator. It is a sacred digital pilgrimage experience for women entering Sheetal Kandola's Shakti Shala ecosystem.

The app helps a seeker ascend a symbolic Himalayan mountain path through Tantra micro-teachings, moon rhythms, goddess thresholds, and somatic reflections. The experience should feel like a quiet mobile sanctuary, not a social app or dashboard.

## Core Product Thesis

Shakti Shala is a mobile pilgrimage app where the Shakti Shala Sherpa helps women ascend a sacred mountain path through Tantra teachings, moon rhythms, goddess thresholds, and somatic reflections — guiding each seeker toward the doorway her system can actually hold.

## Key Language

Use this exact product line:

```text
The Shakti Shala Sherpa helps you ascend the mountain.
```

Use this canonical Sherpa line:

```text
You do not need to climb the whole mountain today.
```

## Important Role Distinction

The Sherpa is not a guru.
The Sherpa is not Sheetal.
The Sherpa is not a therapist.
The Sherpa is not an oracle.
The Sherpa is not the goddess.
The Sherpa is not the destination.

The Sherpa supports ascent, protects pace, reads terrain, and recommends the next safe doorway.

## Visual Direction

Design a premium, mobile-first, sanctuary-like experience.

Visual world:

- Himalayan mountain at dusk/night
- dark stone temple threshold
- spiral mountain path
- ember light
- muted gold
- deep burgundy
- charcoal
- clay/rust
- soft mist
- candle glow

The app should feel:

- grounded
- feminine
- premium
- quiet
- embodied

- mysterious but readable
- like a sacred expedition

Avoid:

- neon gradients
- cyberpunk AI

- oversexualized tantra imagery

- noisy social feeds

## App Structure

Create the following screens/components:

### 1. Opening Threshold

Copy:

```text
Shakti Shala

A mountain path for practice, rhythm, shadow, and Shakti.

You do not need to climb the whole mountain today.
Begin with the next steady step.
```

CTA:

```text
Begin the Path
```

### 2. Meet the Sherpa

Copy:

```text
Your Shakti Shala Sherpa helps you ascend the mountain.

The Sherpa is not a guru. The Sherpa reads the terrain, protects the pace, and helps you choose the next safe doorway.
```

CTA:

```text
Continue
```

### 3. Entry Discernment Quiz

Ask these questions one at a time:

1. What are you carrying right now?
   Options:
   - Too much responsibility
   - A threshold I cannot name
   - A desire for deeper practice
   - Grief, rage, or shadow
   - A longing for retreat or immersion

2. What pace feels supportive?
   Options:
   - Gentle and ongoing
   - Personal and focused
   - Structured and committed
   - Immersive, but only with readiness

3. What are you longing for?
   Options:
   - Rhythm
   - Clarity
   - Shadow integration
   - Community
   - Retreat preparation

4. What kind of support feels sustainable?
   Options:
   - Short practices
   - Weekly circle
   - 1:1 support
   - A deeper container
   - Retreat readiness

### 4. Mountain Map

Show a vertical or spiral mobile map with these gates:

- Base Camp
- Threshold
- ABC Path
- Moon Gate
- Durga Gate
- Kali Gate
- Lakshmi Gate
- Shakti Shala

Some gates may appear softly locked or dimmed, but avoid punitive language. Use protective language.

Sherpa note:

```text
The mountain is not climbed by force. Your path opens through rhythm, practice, and readiness.
```

### 5. ABC Path

Show five Tantra micro-lessons:

- A — Awareness
- B — Breath
- C — Chaitanya
- D — Dharma
- E — Embodiment

Each lesson card should include:

- micro-teaching
- reflection
- body practice
- journal prompt
- Sherpa note

### 6. Moon Gate

Create a New Moon / Full Moon rhythm challenge.

Flow:

```text
Set intention
→ Practice for 3 days
→ Reflect
→ Release
→ Receive next prompt
```

Sherpa note:

```text
Rhythm is how the body learns to trust the path.
```

### 7. Goddess Gates

Create three MVP goddess gates.

#### Durga Gate

Theme: foundation, protection, boundaries, steadiness.

Sherpa note:

```text
Durga protects the path before the deeper gates open.
```

#### Kali Gate

Theme: shadow, truth, death-rebirth, cutting illusion.

Sherpa note:

```text
The Kali Gate is not closed. It asks for preparation.
```

#### Lakshmi Gate

Theme: receiving, worth, nourishment, beauty, value.

Sherpa note:

```text
Receiving is also a practice of steadiness.
```

### 8. Pathway Reveal

Recommend one primary pathway based on the quiz answers. Keep logic simple.

Possible pathways:

- Open Gate
- Moon Rhythm
- Weekly Circle
- 1:1 Support
- 6-Week Container
- Retreat Readiness
- Temple Library

Use this pattern:

```text
Your path points toward [DOORWAY].

Sherpa Note:
[Short reflection based on readiness signals.]

Next step:
[CTA]
```

### 9. Save My Path

Show contact capture only after the pathway reveal.

Fields:

- First name
- Email
- Optional WhatsApp

CTA:

```text
Send My Path
```

### 10. Rest Point

End with:

```text
This threshold is complete.
Rest before ascending.
```

## Interaction Rules

- Make it feel mobile-first.
- Use a calm, premium interface.
- Use progressive reveal.
- Avoid clutter.
- Use short copy on each screen.
- Use the Sherpa as a small note or companion card, not a chatbot window.
- Do not make the Sherpa too cute or cartoonish.
- Use soft gates and protective locks.
- Make the mountain map the emotional center of the app.
- Keep the app lightweight enough to export as a working React/Vite prototype.

## Suggested Technical Output

Generate a working React/Vite prototype with:

- componentized screens
- mobile-first responsive design
- simple local state for quiz answers
- simple pathway recommendation logic
- no backend
- no payment system
- no auth
- no external API requirement
- elegant CSS animations only if lightweight

## Final Reminder

The app should not gamify spirituality.
It should make the sacred pathway visible.

The goal is not addiction.
The goal is readiness.
