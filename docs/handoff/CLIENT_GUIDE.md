# Client Guide

## What Exists Now

Shakti System OS has a sanctuary app foundation, repo hygiene, integration boundaries, and a documented path toward Drive-backed Vault intelligence.

## What The Client Should Use

- Google Drive for source assets and production files.
- The app for seeker-facing experience.
- Airtable later for metadata, readiness, access, and retreat operations.
- Notion later for team-facing dashboards and decisions.

## What Not To Do Yet

- Do not enter secrets into GitHub.
- Do not treat the app as the source file library.
- Do not publish unreviewed Vault material into the public experience.
