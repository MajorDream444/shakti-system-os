# Admin Guide

## Admin Responsibilities

- Keep Drive organized.
- Review source assets before Airtable mapping.
- Keep Airtable records clean and approved.
- Track retreat readiness with human review.
- Keep secrets out of GitHub.

## Asset Review

Use:

```text
docs/handoff/UNTRACKED-ASSETS-REVIEW.md
```

## Backend Prep

Before live sync:

1. Confirm Drive root folder.
2. Confirm Airtable base.
3. Confirm table names and required fields.
4. Confirm who reviews asset metadata.
5. Confirm what content is safe to expose in the app.
